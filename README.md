best served: with [ai-digest](https://github.com/khromov/ai-digest)

1. set ur sys prompt as  [system prompt](/prompts/prompt.md) 
2. give the ai txt file obtained from the [ai-digest](https://github.com/khromov/ai-digest)
3. construct ur prompt like on [request example](/prompts/user-request-format-example.md)
4. see if yall get issues with the triple backticks, codeblock separation, fix it
5. if all g, copy as, click triple dot -> copy as markdown
6. ctrl + shift + p -> `Paste AI markdown as operation`
